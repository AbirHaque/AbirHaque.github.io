Since I started programming, I came across a fair amount of IDEs, but Repl.it was profound in developing my passion in programming. I would spend hours after school fiddling around with what Repl.it offered. Before I present my review, allow me to disuss the features Repl.it has to offer.

Repl.it is an in-browser IDE that promotes both beginner and professional programming. Repl.it's IDE promotes beginners as it offers:
- VSCode intelligence, which offers parameter hinting, autoformatting of code, definition jumping, and additional features listed here
- A simple file manager
- An additional command line interface
- Example code when creating new files
- A run button simplifying the compilation/interpretation/execution process

Repl.it's IDE also promotes professionals, as it offers:
- A personal virtual machine
- Database environment for each project
- A large selection of packages
- Github integration
- Support for over 50 languages
- A feature called Multiplayer, a feature to code with a team on the same file

In addition, Repl.it offers a community of developers, as the site has a developer platform. The platform offers:
- A developer forum
- Several project templates
- A large selection of programming tutorial pages
- Community lead classroom environments
- Online code jams that challenge developers to create projects based on given themes. Prizes include from cryptocurrency, money, or temporary free access to the Hacker plan, a plan that will be discussed later in the review.

The paid account version that Repl.it offers is the Hacker plan. Before I delve into what the Hacker plan offers, I must discuss the Starter plan, Repl.it's free plan tier. Aside from what has already been mentioned, the Starter plan offers:
- 500 MB storage
- 500 MB memory
- 0.2 - 0.5 vCPUs

The Hacker plan offers:
- 5 GB of storage
- 2 GB of memory
- 2 vCPUs
- Private projects.

With context of Repl.it's features given, I rate Repl.it an 8/10.

I truly enjoy how Repl.it offers many features geared towards both beginner and professional niches of programming. Repl.it is also maintained by a tightly knit, dedicated group of developers, which is another plus, as it results in a much more stable platform. I also believe that Repl.it is a wonderful platform to use when trying out a new language, as the site offers both language support and tutorials of a great deal of languages. In addition, Repl.it's online IDE makes it convenient for developers programming on various computers and operating systems. Overall, Repl.it makes programming convenient and productive. Repl.it is encourages productivity to the point that I finished my sophmore year computer science curriculum a half-semester early by doing all my homework and projects there! I also plan on using Multiplayer on Repl.it a whole lot in the near future. Multiplayer makes programming is small groups extremely convenient, especially with the ongoing pandemic keeping us indoors.

One major downside of Repl.it is its offline incapability, which tackles my argument of Repl.it being convenient. However, the Repl.it developer team is working on getting offline support soon, which certainly deserves high praise! A slight downside is the little storage space and memory offered by the Starter plan. I call this a slight downside as system resource usage limitations are not going to be a problem for developers merely trying out languages. Developers intending to use Repl.it to build large applications should look to the Hacker plan. Repl.it, overall, is not great for developers needing large amounts of system resources, which is the main reason why I use Repl.it a bit less than I used to.

In conclusion, Repl.it deserves an 8/10 as it offers online convenience at the expense of limited system resources and no offline support. If you want to try out a new language, host a small website, or code in real time, Repl.it is the best IDE for you. If you need large amounts of system resources for you next project, Repl.it is not the place for you.
