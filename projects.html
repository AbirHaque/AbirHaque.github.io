Projects
Below are a few of my projects that I have completed or are ongoing:





SimpleWaveManager: Javadoc
SimpleWaveManager is library I wrote during April 2021. The library is designed to allow mono/stereo audio input/output code to be portable across different Java versions (Java 6 and higher), specifically those without access to JavaX Sound or Android Media libraries. This library is utilized in Interfere Free Calc.





Interfere: Home Page
The Interfere Software collection is a series of digital audio editors that I have been working on since October 2020. Interfere Free Calc, a minimal audio player and visualizer for mathematical equations, is the most recent program within this collection. Interfere Cascade, a MIDI composition editor, was the first program. Here is a short song I composed entirely with Cascade!





CSVOL: Product Code Documentation Page
Character Separated Value Organizer Language, or CSVOL (\kasväl\), is programming language created for the 2020 Repl.it Programming Language Jam. Development of CSVOL started since August 2020 and ended December 2020.





Tallia: Product Code
Development of this app lasted from July 2020 to September 2020. Tallia is was developed in partership with NIR-V. The app's current functionalities include: keeping daily entries of how a user feels, graphing the ups and downs a user has each day, and an optional feature of sending daily entries to a guardian through SMS.





SwimDB:
Development of this app started lasted from February 2020 to August 2020. SwimDB was developed in partership with my high school's swim coach. The app is aimed towards swim coaches, as the app serves as a swim team manager. The app's current functionalities include: tracking individual swimmer stats, creating workout sets for up to eight lanes, and automatically adjusting duration and intensity of each workout for individual swimmers. All calculations are based off of an Excel spreadsheet my coach uses to manage his own team.





Bangla Alphabet App: Product Press Mention
Development of this app lasted from September to October 2019. I created the app with two intentions: to have an app that provides interactive Bangla alphabet charts to aide in Bangla literacy and to have a submission for the 2019 KS-04 Congressional App Challenge. While I consider this app to be rudimentary, it was the biggest project I ever completed in 2019.





Top Subbreddit Image Scraper: Product Code
This console project is one of many laidback weekend projects I do for fun. Development of this project lasted for one weekend in September. It scrapes a Reddit page for the highest upvoted image/meme when given a subreddit and filter, then presents a preview image link to the user. This is the complicated way to look at memes on Reddit.
